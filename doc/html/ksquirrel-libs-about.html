ksquirrel-libs can be described in following terms:

  • ksquirrel-libs is a set of codecs for the browser of images KSquirrel.
  • format of libraries is very simple, so you can write the codec necessary to you.
  • you can use ksquirrel-libs in your program - simply realize the mechanism, allowing to manipulate a set of such libraries (like SQ_LibraryHandler class in KSquirrel).
  • each codec has a strictly defined set of functions which provide the interface of library.
  • from the version 0.6.0-pre1 codecs are written on C++.
  • from the point of view of C++ all codecs absolutely identical - they differ only with contents of methods.
